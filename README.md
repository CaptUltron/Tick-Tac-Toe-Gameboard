**TIC-TAC-TOE-GAME**

**Description**
It is the in-famous OG tic-tac-toe where a perfect line wins the game and none results in a draw

**Features**
1. Two players with symbols 'X' and 'O'
2. Players name are editable, by default it is Player 1 and Player 2
3. It contineously displays the score as the game starts
4. On achiving the perfect combination, Game Over dialog box pops up with name of the player who won
5. If all squares selected but nobody won, Game Over dialog box pops up with "Its a draw message"
6. Game Over dialog box offers a rematch button that resets the game

<h2>Development</h2>
To get a local copy of the code, clone it using git:

```
git clone https://github.com/CaptUltron/Tick-Tac-Toe-Gameboard.git
cd Tick-Tac-Toe-Gameboard
```

Install dependencies:

```
npm install
```

Now, you can start a local web server by running:

```
npm run dev
```

and then you can open http://localhost:5173 to view it in the browser.
